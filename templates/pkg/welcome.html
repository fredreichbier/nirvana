Welcome!

At the ooc nirvana, you can browse package usefiles and even upload your own.

If you want to download packages from nirvana, make sure you have added the gpg public key first:

curl http://nirvana.ooc-lang.org/nirvana.asc > nirvana.asc
reincarnate add-key nirvana.asc

Please make sure this key is the same as here. If it is not, please tell us at #ooc-lang on freenode.
